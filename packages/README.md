# Packages

This directory contains reusable Flutter/Dart packages and tools for the project.

These packages manage core systems like data loading, assets, or specific tooling that can be used across multiple apps or independently.

## Packages
* `fodder_assets` - Logic for extracting, loading, and managing the original game assets (e.g., loading map files or `.voc` audio).
* `fodder_tools` - Utility scripts or supplementary build steps that might be used to pre-process game assets.
