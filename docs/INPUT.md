# Input Reference

---

## Mouse

| Input | Context | Effect |
|-------|---------|--------|
| **Left-click** | In mission | Move the selected squad toward the cursor (sets a walk-target). |
| **Left-click** | Map overview | Dismiss the overview and return to gameplay. |
| **Left-click** | No squad selected | Click a soldier/squad to select it. |
| **Right-click** (hold) | In mission | Fire the current weapon toward the cursor. |
| **Right-click** (first press) | In mission | Overrides the squad leader's fire cooldown (allows immediate shot). |
| **Left + Right** (simultaneously) | In mission | Fire special weapon (grenade or rocket); the non-special weapons are disabled while both buttons are held. |
| **Cursor at screen edge** | In mission | Scrolls / pans the camera in that direction. Camera auto-pans toward the squad leader when the cursor is not at an edge. |
| **Left-click** | In vehicle | Steer the vehicle (or helicopter) toward the cursor position. |

---

## Keyboard — In-Mission

All keys During active gameplay, not during the map overview.

| Key | Effect |
|-----|--------|
| **P** | Toggle pause. |
| **M** | Show full-level tactical map overview (mini-map). |
| **Space** | Cycle weapon (grenades → rockets → bullets). |
| **1** | Select squad 0. |
| **2** | Select squad 1. |
| **3** | Select squad 2. |
| **Escape** | Abort mission — first press pauses; press again to fully exit. |

---

## Keyboard — Cheat Keys

Gated behind `DEBUG` builds.

| Key | Effect |
|-----|--------|
| **F9** | Toggle invincibility for the currently selected squad. |
| **F10** | Skip the current phase / mission. |
